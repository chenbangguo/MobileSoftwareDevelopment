一、名片展示的个人信息
    个人头像 / Logo：页面视觉中心，圆形展示
        姓名：主标题文字，大号加粗显示
        职位：软件设计师
        联系方式
        手机号码
        社交账号
        邮箱地址

二、布局结构 & Composable 说明
    整体采用 Column 垂直布局，分为上下两个功能区域：
    上半部分（个人信息区）
        外层：Column
        内部组件：Image（头像）+ Text（姓名）+ Text（职位）
        作用：展示个人身份信息，整体水平居中
    下半部分（联系方式区）
        外层：Column
        内部：多行 Row 组合
        每行结构：Icon（图标）+ Spacer（间距）+ Text（文字）
        作用：展示联系方式，整体居中、内部左对齐
    核心组件
        Column：垂直方向排列组件
        Row：水平方向排列组件
        Image：显示头像图片
        Icon：显示系统图标
        Text：显示文字信息
        Spacer：控制组件间距
        Modifier：设置大小、边距、居中、背景色、圆形裁剪

三、遇到的问题 & 解决过程
    1. 底部联系方式排版问题
        问题：内容无法居中、对齐混乱
        解决：调整外层 Column 居中，内部 Row 左对齐，统一间距
    2. 头像改为圆形
        问题：默认头像为方形，不美观
        解决：使用 Modifier.clip(CircleShape) 将图片裁剪为圆形
    3. 代码编译报错
        问题：fillSize、mod 等方法不存在
        解决：修正为标准 API fillMaxSize()、modifier
    4. 界面样式优化
        问题：背景颜色过于厚重
        解决：更换为淡粉色、高透明度背景，搭配柔和文字与图标颜色，提升视觉效果
        
四、实验总结
本次实验使用 Jetpack Compose 完成了电子名片界面开发，掌握了 Column、Row 基础布局的嵌套使用，学会了图片圆形裁剪、界面样式修饰、UI 排版优化等技能。在开发过程中解决了编译错误、布局对齐、颜色搭配等问题，最终实现了一个简洁美观、布局整齐、可正常运行的电子名片应用。
